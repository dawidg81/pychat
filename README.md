# pychat
pychat is a simple chat software written in [Python](http://www.python.org/). The project is including both server software as well as a client.

**DISCLAIMER!** My project pychat and its name *pychat* are not associated with [this](https://github.com/akoidan/pychat), [this](https://pypi.org/project/PyChat/) or even [this](https://play.google.com/store/apps/details?id=com.code.pychat&pli=1). I've came with the name *pychat* not knowing about these other projects until I have searched them up.
